Throttle is a minimal yet powerful To-Do List web app designed to help you stay consistent with your daily goals. It’s built for speed, simplicity, and motivation.

With an easy-to-use interface, users can quickly add tasks, mark them as complete, and manage their workflow without distractions. The project is designed with a productivity-first mindset and focuses on clean UI and smooth interactions.

Future enhancements include streak systems, progress tracking, and smarter task management features.

Tech Stack: HTML, CSS, JavaScript
Built to improve discipline, one task at a time ⚡
